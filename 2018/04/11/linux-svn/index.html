linux install svn

发表于 2018-04-11

1.yum install svnserve

2.create svnrepos

3.create svn 版本库

4.modify config file

authz：负责账号权限的管理，控制账号是否读写权限
passwd：负责账号和密码的用户名单管理
svnserve.conf：svn服务器配置文件
modify authz

在文件末尾添加 用户名和读写权限 rw read write []中的斜杠,一定是反斜杠

modify passwd

modify svnserve.conf

#authz-db = authz 要注释掉 不注释掉 会出现svn 认证失败
5.防火墙开启

6.启动svn服务器
bash svnserve -d -r /var/svn/svnrepos ps -ef|grep svn kill -9 端口号bash
7.客户端访问svn
输入地址：svn://ip:3690/salary
